This demo features distinct classes:
  • The com.pump.text.UnderlineHighlightPainter class is a javax.swing.text.Highlighter class that, well, underlines text. Our original use for this was to highlight misspelled words or syntax errors.
  • The com.pump.text.TextBoxHighlightPainter is a fun one. This uses the com.pump.awt.TextBlock class to render a big in-your-face highlight.
  • The com.pump.text.WildcardPattern class is by the far the most interesting class in this demo, but it's a behind-the-scenes util class. It implements a wildcard expression as defined here. Basically this is intended to be simpler/leaner than than a regex pattern: you can use *'s, ?'s and square brackets.
