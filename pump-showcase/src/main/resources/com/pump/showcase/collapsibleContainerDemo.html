Call it an accordion, call it a disclosure control, or call it progressive. It's a great alternative to tabs when you have too many UI controls to squeeze into your available real estate.

The CollapsibleContainer helps set up a series of labeled sections. Through client properties you can control whether a section is collapsible, and/or what vertical weight a section should have in the layout.

Scrollpanes preserve their existing rectangular border. Trees and lists are given a single-pixel rectangular border. Everything else is put inside a box UI to group controls together.
