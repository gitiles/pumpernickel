Once upon a time we needed a widget to edit angles. Technically the javax.swing.JSlider had all the data we needed (a value that had a minimum and maximum), but using a traditional slider to manage an angle was unintuitive.

The AngleSliderUI is a new SliderUI that renders the value of slider degrees on a circular dial. This was inspired by a widget we saw in iWeb Back in those days Aqua still had elements that appeared to resemble candy (bubbly, plastic, shiny). Nowadays both Apple and Microsoft are trending more towards a minimalist design, though.

In this app you can see an example of this UI in the Strokes, MouseSmoothing demo. Originally we used it to edit drop shadows in an image editor.
