Transition3D Demo

What Is It

The Transition3D class helps support transitions modeled with simple 3D transformations. For now (and the foreseeable future) there are only two subclasses:

  • CubeTransition3D rotates images as if they are on a cube.
  • FlipTransition3D flips images as if they are on opposite sides of a sign.

How To Use It

Like the Transition2D classes the usage is very simple:

CubeTransition3D t = new CubeTransition3D(Transition.DOWN, false);
float progress = ...;
t.paint(graphics2D, imgA, imgB, progress);

How It Works

These use the ImageContext class to render an image using four transform points. Java's built-in AffineTransform class only lets you use three transform points (not four), so it is more limiting.
