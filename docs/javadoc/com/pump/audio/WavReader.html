        This method fixes a file with corrupt size fields.

        A RandomAccessFile is used to redefine the size of the "RIFF" and "data" chunks based on the real file size.

        Throws:
        java.io.IOException - if an IO problem occurs.

      • isValidSizeHeader

        public static boolean isValidSizeHeader(java.io.File wavFile)
        This validates the first header of a wave file.

        This method was written in response to QTJ's MovieExporter occasionally converting MP3s to a wave file incorrectly. The resulting wave file contained all the correct PCM data, but the chunk size of the "RIFF" and "data" chunks are zero.

        This method only validates the first header. If this method returns false, the method fixCorruptSizeFields() should fix the problem.

        Parameters:
        wavFile -
        Returns:
        true if the the "RIFF" size header is consistent with the real file size.
