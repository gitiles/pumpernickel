    public abstract class AbstractShapeTransition2D
extends Transition2D
    This takes any abstract silhouette and will zoom it in/out to reveal the new frame.

    This class is actually a little bit clever, because it will study exactly to what proportion it has to zoom the silhouette to completely cover the frame size.
