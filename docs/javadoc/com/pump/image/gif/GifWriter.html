        public GifWriter(java.io.OutputStream out,
                 java.awt.Dimension size,
                 java.awt.image.IndexColorModel globalColorModel,
                 int loopCount,
                 int backgroundColorIndex,
                 GifEncoder customEncoder)
          throws java.io.IOException
        Constructs a GifWriter with a global color table.
        Parameters:
        out - the output stream to write to. This object will not close the output stream when it is finished.
        size - the bounds of this animation. Images larger than this value will be cut off. (Images smaller than this value will have dead space surrounding them.)
        globalColorModel - a byte-based IndexColorModel to use as a global color palette.
        loopCount - how many times this GIF file should loop. If this is zero (or negative), then this GIF will not loop. If this is greater than 65535, then this GIF will loop forever.
        backgroundColorIndex - the index in the global color table to use as a background color
        customEncoder - an optional encoder. If null (which is encouraged), then GifEncoderFactory.get().createEncoder() is used.
        Throws:
        java.io.IOException - if the underlying OutputStream has trouble writing any of the GIF header information.
