Class AbstractAttributeDataImpl

  • java.lang.Object
    • com.pump.data.AbstractAttributeDataImpl
  • All Implemented Interfaces:
    java.io.Serializable
    Direct Known Subclasses:
    AttributeDataImpl

    public class AbstractAttributeDataImpl
extends java.lang.Object
implements java.io.Serializable
    This class provides the functionality of an AttributeData, but most of the methods are protected.

    The AttributeDataImpl will make these methods public, but some other subclasses may want to more tightly control exposure to some methods. For example: maybe external classes cannot call get/set for any arbitrary property. In some subclasses we can control with explicit methods exactly which attributes can be set.
