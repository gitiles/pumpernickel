Class ButtonIcon

  • java.lang.Object
    • com.pump.icon.button.ButtonIcon
  • All Implemented Interfaces:
    javax.swing.Icon
    Direct Known Subclasses:
    MinimalDuoToneCloseIcon

    public abstract class ButtonIcon
extends java.lang.Object
implements javax.swing.Icon
    This is an icon that can change colors based on a set of input variables (rollover, enabled, pressed, etc.).

    If you use the constructor that accepts a button, then this is all automatically handled for you. Whenever a visual change is appropriate a 100ms animation will fade in the new color scheme.
