Package com.pump.icon.button

  • Interface Summary 
    Interface Description
    ButtonIconColors
    This translate ButtonState information into a map of colors.
  • Class Summary 
    Class Description
    ButtonIcon
    This is an icon that can change colors based on a set of input variables (rollover, enabled, pressed, etc.).
    ButtonIconManager
    This connects an AbstractButton with a ButtonIcon and manages the required listeners and animations.
    ButtonState
    This provides a snapshot of a button's state (is it pressed, enabled, selected, focused, etc.).
    MinimalDuoToneCloseIcon
    This close icon resembles the close icon in Safari's tabs.
