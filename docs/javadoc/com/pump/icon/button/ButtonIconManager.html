        Parameters:
        button - the button to listen to (for rollover, focus changes, enabled changes, etc.), and to assign the icon of.
        icon - the ButtonIcon to use. This icon is assigned to the button, and this manager constantly calls ButtonIcon.setColors(Map) and repaints the button as needed.
        colors - this translates button state information into colors.
        properties - an optional set of properties that will be stored in each ButtonState. This lets you add custom attributes that are not already defined a ButtonState.
