Class BasicMouseSmoothing

  • java.lang.Object
    • com.pump.geom.BasicMouseSmoothing
  • All Implemented Interfaces:
    MouseSmoothing

    public class BasicMouseSmoothing
extends java.lang.Object
implements MouseSmoothing
    This collects a series of points and creates a smooth curve.

    For example, if the user drags the mouse and a shape is made from those points it will appear jagged and unappealing. This class will collect those points and return nice curved shapes. Some points will be "dropped" for the sake of creating a smoother curve.
