      • guessBehavior

        MissingJarResponse guessBehavior(java.io.File jarFile)
        This method guesses the desired behavior.

        If getBehavior() has already returned null then we're going to show a dialog to the user to make them decide. But this method helps prepopulate the choices. If the user likes what they see, they can just hit the return key and move on.

        This method may consult other projects and see if they have already stored a Behavior for this file.

        Parameters:
        jarFile - the file to guess the Behavior for.
        Returns:
        a reasonable guess at the Behavior to associate with this File, or null.

        Parameters:
