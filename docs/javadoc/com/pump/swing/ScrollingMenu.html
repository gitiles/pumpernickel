Class ScrollingMenu

  • java.lang.Object
    • com.pump.swing.ScrollingMenu

  • public class ScrollingMenu
extends java.lang.Object
    This takes a collection of menu items and makes a scrolling menu.

    As of this writing: it looks like if a popup menu is too large then Swing/AWT does not help the user navigate it.

    This is loosely based on/inspired by darrylbu.util.MenuScroller. I had trouble sometimes with the placement of that popup menu, though, so this implementation separates the JMenu (which has to constantly be revised as you scroll menu items) from the list of menu items.
