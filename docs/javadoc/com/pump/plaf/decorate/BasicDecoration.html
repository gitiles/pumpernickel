Class BasicDecoration

  • java.lang.Object
    • com.pump.plaf.decorate.BasicDecoration
  • All Implemented Interfaces:
    DecoratedListUI.ListDecoration, DecoratedTreeUI.TreeDecoration, ListTreeDecoration

    public class BasicDecoration
extends java.lang.Object
implements ListTreeDecoration
    This is a simple 3-state TreeDecoration and ListDecoration that stores a fixed "normal", "rollover" and "pressed" icon state.

    The default implementation of isVisible(..) will only render this decoration when a tree node is selected. This is just an aesthetic choice you are free to override. (The original motivation for this decision was: too many decorations might become visual clutter.)

    It is assumed that if a rollover or pressed icon is provided that it will be the same size as the normal icon.
