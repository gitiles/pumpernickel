Class ShapeCreationUI

  • java.lang.Object
    • javax.swing.plaf.ComponentUI
      • com.pump.plaf.ShapeCreationUI
  • Direct Known Subclasses:
    CubicPathCreationUI, CurvedPolylineCreationUI, MouseSmoothingShapeCreationUI

    public abstract class ShapeCreationUI
extends javax.swing.plaf.ComponentUI
    A UI for a ShapeCreationPanel.

    This abstract superclass offers a few common functions, but subclasses most implement specific MouseListeners and implementation of each handle.

      • validate

        protected java.awt.Shape validate(java.awt.Shape shape)
        This converts the argument to a shape compatible with this UI.

        For example: the CurvedPolylineCreationUI only wants to work with CurvedPolylines. This may necessarily lose some shape data in making it conform to the expected format.

        The default implementation simply returns the argument.

        protected java.awt.Stroke getStroke(ShapeCreationPanel panel,

        Return the stroke used to paint a WritingStroke. This stroke is independent of the panel's AffineTransform. (So if the stroke is expected to scale with the transform, then this method needs to consult the AffineTransform and change accordingly.)

      • paintShape

        protected void paintShape(java.awt.Graphics2D g,
                          ShapeCreationPanel panel,
                          int shapeIndex,
                          java.awt.Shape transformedShape)
        This paints an individual shape. Subclasses may override this to control the color/stroke of shapes.

        The default implementation here paints a 2-pixel focus ring around a shape if it is the currently selected shape.

        See Also:
        getStroke(ShapeCreationPanel, int)

      • paintControls

        protected void paintControls(java.awt.Graphics2D g,
                             ShapeCreationPanel scp)
        Paint handles/controls above the shape data.

        When implementing this method: note that this is always invoked, and you need to consult the ShapeCreationPanel to determine when handles are supposed to be active.
