      • installTweenEffect

        public static void installTweenEffect(javax.swing.JComponent jc,
                                      java.lang.String targetPropertyName,
                                      java.lang.String realPropertyName,
                                      float increment,
                                      int updateInterval)
        Tween a JComponent client property towards a target value.

        This is unrelated to UIEffect objects, but the intention is so similar that it seemed appropriate to keep this static method here.

        Parameters:
        jc - the component the values are stored in.
        targetPropertyName - the target property. This might oscillate between a few fixed values (like 0 and 1).
        realPropertyName - the actual property. Once the target is modified: a looping javax.swing.Timer is used to continually change this value by increment until that target is reached.
        increment - the amount to adjust the real property by every iteration.
        updateInterval - the number of milliseconds between iterations. So if increment is .05 and updateInterval is 200: then the real property will change .05 units every 200 ms until the target is reached.
