com.pump.plaf

Class AbstractSearchHighlight

  • java.lang.Object
    • com.pump.plaf.AbstractSearchHighlight
  • Direct Known Subclasses:
    AquaSearchHighlight, FirefoxSearchHighlight, NudgeSearchHighlight

    public abstract class AbstractSearchHighlight
extends java.lang.Object
    This renders a temporary effect somehow animating a selection in a JTextComponent. This assumes the text component is in a JFrame, JApplet, or other RootPaneContainer because it renders the animation with the help of the LayeredPane.

    This creates images of the selection when constructed, and renders those images through JComponents. Once the image is drawn it cannot be modified, but you can modify it's opacity and transform through the updateAnimation method.

    Also you can modify the duration of the animation by calling UIManager.put("SearchHighlight.duration",new Integer(millis)), or overriding the getDuration() method of this object.

    This is automatically created by calling TextSearch.highlight().

        protected abstract void updateAnimation(javax.swing.JComponent[] highlights,
                                        float fraction)
        This is constantly called while the animation is playing. It gives you an optional chance to modify the highlights a little bit.
        Parameters:
        highlights - the components to modify. You should change the client property "opacity" (to be a number from [0, 1]) and "transform" (to be an AffineTransform) to modify these highlights.
        fraction - a fraction from [0, 1], indicating how progressed this animation is. After t = 1 this method is no longer called and this highlight will fade out.
