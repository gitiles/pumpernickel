      • browseFile

        protected java.io.File browseFile(Document doc)
        Browse for a file to save a Document to.

        This method is called when this action is invoked but the document's Document.getFile() returns null.

        This method uses the file extension provided in the constructor. The default implementation of this method will throw a NPE if that file extension is null.

      • save

        public void save(Document doc,
                 java.io.File file)
          throws java.lang.Exception
        Save a document to a given file. If successfully this should also fire PropertyChangeListeners for the Document.PROPERTY_FILE property if that property changed.

        This method can be called on any thread. If it is called on the EDT then an interdeterminate cancellable progress dialog is shown that blocks the EDT while another thread calls doSave(Document, File, Cancellable).

        Throws:
        java.lang.Exception
