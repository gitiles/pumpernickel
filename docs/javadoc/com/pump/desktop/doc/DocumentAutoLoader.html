com.pump.desktop.doc

Class DocumentAutoLoader

  • java.lang.Object
    • com.pump.desktop.doc.DocumentAutoLoader

  • public class DocumentAutoLoader
extends java.lang.Object
    This saves a list of open documents and re-loads those documents in new sessions.

    The default implementation relies on a OpenDocumentAction, so that needs to be added to a DocumentControls before this object is instantiated.

      • deserialize

        protected Document deserialize(java.lang.String value)
        Convert a String created by serializeDocument(Document) back into a Document.

        Presumably the original String was created in a previous session/JVM, so this needs to be something that makes sense across sessions.

        The default implementation assumes the String is a filepath. If that file no longer exists, then this returns null.

        Returns:
        a Document from the serialized representation, or null if the argument isn't recognizable or usable.
