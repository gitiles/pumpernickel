Package com.pump.graphics

  • Interface Summary 
    Interface Description
    DrawInstruction
    This instruction paints the stroke of a shape.
    FillInstruction
    This instruction paints the fill of a shape.
    GraphicInstruction
    An instruction to paint something to a Graphics2D.
  • Class Summary 
    Class Description
    AbstractGraphics2D
    This implements several methods to get Graphics2D's off the ground.
    BaselineGraphics2D
    This simply records the baselines that text is drawn at.
    BasicDrawInstruction
    The default implementation of a DrawInstruction.
    BasicFillInstruction
    The default implementation of a FillInstruction.
    BasicShapeInstruction
    This paints a FillInstruction and then a DrawInstruction on top of it.
    DualGraphics2D
    This delegates to two underlying Graphics2Ds, so you can send the same drawing instructions to multiple destinations.
    FlattenTextGraphics2D
    This filter converts all calls to render text into shape-based instructions.
    GraphicsWriter
    This object provides a simple way to model/export layered vector graphics.
    GraphicsWriterDebugger
    A simple debugger tool for graphics.
    ImageInstruction
    An instruction to draw an image.
    TextBoxInstruction
    A GraphicInstruction for text boxes.
    TextOnlyGraphics2D
    A Graphics2D that only delegates methods that might relate to drawing text.
