The Pumpernickel project is a library of open-source code covering a wide range of topics. I'd love to tell you that there's a singular purpose or mission statement driving everything in this project, but the reality is: this is a collection of classes and tools that I've worked on over the years for a variety of projects and needs.

Please feel free to browse the javadoc and the repository and to email me directly with feedback or suggestions.

Everything in the com.pump codebase is released here under a MIT license.

If you want a quick jump start into some of the code covered here, download the latest showcase jar and open it. This includes some of the obvious splashy stuff like color dialogs and multithumb sliders, but it also includes some performance tests that present alternatives to certain ImageIO functions or java.awt.geom.Area performance problems.
